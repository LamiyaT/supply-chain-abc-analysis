
# ABC Analysis for Supply Chain Optimization

2️⃣ Project Description

This project performs ABC Analysis on supply chain data 
based on revenue generated.
Products are classified into A, B, and C categories 
using cumulative percentage.


3️⃣ Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

4️⃣ Steps Performed

1. Data cleanin
2. Calculating percentage
3. Calculating cumulative percentage
4. Classifying into A, B, C categories
5. Visualization

5️⃣ Results

- A category items contribute ~80% revenue
- B category items contribute ~15%
- C category items contribute remaining revenue
